Detailed Description

MQTT connection test with multiple servers - Pico W.

Connect to multiple MQTT server using the LwIP MQTT Client (https://github.com/stm32duino/LwIP/blob/main/src/lwip/apps/mqtt.h , https://www.nongnu.org/lwip/2_0_x/group__mqtt.html , https://docs.oasis-open.org/mqtt/mqtt/v3.1.1/os/mqtt-v3.1.1-os.html)

Attention
Update MEMP_NUM_SYS_TIMEOUT and MQTT_REQ_MAX_IN_FLIGHT in lwipopts.h
Author
GrowBoxGuy - https://sites.google.com/site/growboxguy/raspberry-pi-pico-w
