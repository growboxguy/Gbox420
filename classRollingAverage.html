Detailed Description

Rolling average tracker for smoothing sensor readings. Supports float and int values only.

< RollingAverage class: For smoothing sensor readings

The MovingAverageDepth defines how many previous readings to keep. (!Memory intense!)

Keeps the last few readings and calculates an average.

GrowBoxGuy
Version

Warning
Memory intense! Use it only where really necessary (Instable/noisy sensor readings)
